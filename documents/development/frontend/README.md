# 前端培训文档

## 安装依赖包

```bash
# offline
tar xf node_modules.tar.xz
# online
yarn install
```

![dashboard.png](dashboard.png)

## 开发模式启动 [dashboard](http://localhost:3000)

```bash
yarn start
```
